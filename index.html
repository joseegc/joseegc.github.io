A Tecnologia de Design no CSS

O CSS (Cascading Style Sheets) é uma linguagem muito útil e proeficiente quando estamos falando de Desenvolvimento Web e Design, pois permite uma maior estilização de sites e o uso de layouts pré-prontos, os famosos templates. A ferramenta é conhecida desde 1994 quando urgiu em programas de computadores a necessidade de criar códigos mais visuais e estilizados, como por exemplo na formação de tabelas. Com o CSS você pode manipular:

  • Margens
  • Cores
  • Animações
  • Posições
  • Formatações
  • Listas
  • Sessões/Div's
  • Imagens
  • Sombras

Aplicações do CSS

A linguagem funciona como um apoio a códigos HTML, podendo ser utilizada por mais de uma página ao mesmo tempo, para ser referenciada corretamente no HTML e aplicar suas configurações utiliza-se a seguinte expressão:


< link rel="stylesheet" href="exemplo.css" media="all"/ >
